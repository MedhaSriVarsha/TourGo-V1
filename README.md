# 🌍 TourGo

TourGo is a smart tourism platform designed to help travelers discover popular attractions as well as hidden gems. The platform provides location details, travel information, and an easy-to-use interface for planning memorable trips.

## 🚀 Features

- 🗺️ Explore famous and hidden tourist destinations
- 📍 Detailed information about each location
- 🔍 Search and browse destinations
- 📱 Responsive design for desktop and mobile
- 🎨 Modern and user-friendly interface
- ⚡ Fast and smooth navigation
